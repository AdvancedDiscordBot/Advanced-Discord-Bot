# Welcome & Goodbye Plugin

Customizable welcome and goodbye messages with premium canvas-drawn image cards, text embeds, DMs, and config options.

## Features

- **Welcome Messages:** Send personalized welcome messages to a designated text channel.
- **Goodbye Messages:** Send goodbye messages to a designated text channel.
- **Direct Messages (DMs):** Option to DM welcome messages to users.
- **Image Cards:** Stunning, canvas-generated welcome/goodbye image cards with:
  - User's avatar with gradient border and outer glow.
  - Custom linear gradient backgrounds.
  - Server icon and name header.
  - Username and member count with ordinal suffixes.
- **Interactive Configuration:** Simple slash commands for server admins/managers.
- **Preview & Test Commands:** Test and preview configurations instantly.
