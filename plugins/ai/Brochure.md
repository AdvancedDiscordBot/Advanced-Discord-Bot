# AI Assistant

Official ADB AI assistant plugin. Brings conversational AI to your Discord server.

## Features

- Responds to questions and chat in designated channels
- Context-aware replies within conversation threads
- Mention the bot or use a configured trigger prefix to start a session
- Configurable personality and response style per server

## Setup

After installation, use `/ai setup` to designate which channels the AI monitors. The bot responds when mentioned or when the configured trigger prefix is used.

## Configuration

Adjust the AI model, context window length, and active channels via plugin settings in the dashboard.

## Permissions

Requires `db.read`, `db.write`, `commands.register`, and `events.register`.
